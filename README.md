## Descripción

En este Hito 4, el foco principal fue conectar el sitio web con una API de pizzas.

Para lograrlo, se utilizó el backend entregado como material de apoyo y se subió como **Web Service** en Render. De esta forma, el sitio puede obtener la información de las pizzas desde la API y mostrarla en pantalla.

El frontend también fue desplegado en Render como sitio web.
